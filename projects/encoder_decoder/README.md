# Encoder-Decoder Transformer

该目录只整理 Encoder-Decoder Transformer from scratch 相关版本。当前版本为 [`v0_baseline`](./v0_baseline/)，用于展示英文到中文机器翻译任务上的手写模型、训练闭环、推理闭环、tokenizer 使用和 checkpoint 保存机制。

| 版本 | 目录 | 状态 |
|------|------|------|
| v0_baseline | [v0_baseline/](./v0_baseline/) | 训练进行中 |

当前仓库不包含 Decoder-only from scratch。Decoder-only 后续会单独新建仓库，不属于当前 Encoder-Decoder 仓库。
